# Coffee Shop Website
A modern and responsive **Coffee Shop Website** built using **HTML, CSS, and JavaScript**.  
This project focuses on creating an engaging front-end experience where users can explore the coffee shop’s offerings, view the menu, read customer testimonials, browse the gallery, and get in touch with the shop.

---

##  Project Overview

The Coffee Shop Website is a front-end web development project designed to demonstrate:
- Structured web page layout using **HTML**
- Responsive and attractive UI using **CSS**
- Interactive elements and basic animations using **JavaScript**

The website provides a smooth and user-friendly browsing experience with clean design and intuitive navigation.

---

##  Pages Included

- **Home** – Landing page with an introduction to the coffee shop  
- **About** – Information about the shop, its story, and values  
- **Menu** – Displays coffee and food items  
- **Testimonials** – Customer reviews and feedback  
- **Gallery** – Images showcasing the coffee shop and products  
- **Contact** – Contact form and shop details  

---

##  Features

- Responsive design for desktop, tablet, and mobile devices  
- Clean and modern UI layout  
- Interactive navigation and animations  
- User-friendly interface  
- Simple and readable code structure  

---

##  Technologies Used

- **HTML5** – Structure of the website  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript** – Interactivity and animations
